# 黑潮纪元分卷总纲（动态维护）

> 总目标字数：约100万字 | 8卷 × ~30章 ≈ 240章 | 单章均长约4200字
> 详细大纲见 `master-outline.md`

## 卷一 漂来的人（ch1-30，约12万字）
- **阶段**：起局·从无到有
- **空间**：浮木寨 → 近海
- **三幕**：漂来的人(1-9) → 九号浮桩·死人码头(10-18) → 第一次选择·船团成立(19-30)
- **主角弧**：无名漂客 → 灰雀号船长 → 灰雀船团创立者
- **进度**：第一幕已完成（ch1-9）

## 卷二 雾墓深处（ch31-60，约12万字）
- **阶段**：探索·诡秘线全面展开
- **空间**：近海 → 雾墓海域
- **三幕**：声潮之海(31-40) → 导航员·宁白鸢(41-50) → 遗城入口(51-60)
- **关键**：认知污染机制、宁白鸢登场、"航图是钥匙"、通缉升级

## 卷三 黑帆猎场（ch61-90，约13万字）
- **阶段**：对抗·多势力博弈
- **空间**：骨海 → 黑帆海域
- **三幕**：被猎者(61-70) → 黑帆港(71-82) → 三方角力(83-90)
- **关键**：世界扩展至远洋、教团登场、盐化首次高峰、林仲海之名

## 卷四 沉城遗梦（ch91-120，约13万字）
- **阶段**：深入·身世第一层揭开
- **空间**：雾墓遗城
- **三幕**：下沉(91-100) → 记忆之门(101-112) → 代价(113-120)
- **关键**：林仲海身份确认、林渊"人造人"暗示、盐化生死线、舰灵首次说话

## 卷五 铁灰之海（ch121-150，约13万字）
- **阶段**：转折·联邦真相
- **空间**：铁灰海（联邦控制区）
- **三幕**：北港(121-130) → 档案塔(131-142) → 突围(143-150)
- **关键**：联邦利用黑潮维持权力、罗峻转变、抑盐技术获取、舰灵完整觉醒

## 卷六 裂潮纪（ch151-180，约14万字）
- **阶段**：危机·超级黑潮
- **空间**：全海域
- **三幕**：集结(151-160) → 超级黑潮(161-172) → 余波(173-180)
- **关键**：旧秩序瓦解、林渊成为全域领袖、终极目标锁定海天之门

## 卷七 海天之门（ch181-210，约13万字）
- **阶段**：真相·终极揭秘
- **空间**：裂潮带 → 沉陆海台 → 海天之门
- **三幕**：远征(181-190) → 沉没之都(191-202) → 真相(203-210)
- **关键**：舰灵=林仲海、盐源=海洋免疫系统、林渊=人海桥梁、终极选择

## 卷八 新纪元（ch211-240，约12万字）
- **阶段**：终局·新世界的诞生
- **空间**：整个世界
- **三幕**：对话(211-220) → 涨潮(221-232) → 新纪元(233-240)
- **关键**：人海和解、林仲海消散、黑潮终结、角色终局、"方向是自己的"

- 每新增章节时确认其 `volumeId` 与 `inVolumeOrder`。
- 跨卷转场章节必须在摘要台账中标注"旧卷回收点 + 新卷入口点"。
