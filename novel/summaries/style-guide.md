# 《黑潮纪元》写作风格指南

> 参考猫腻《间客》的写作风格，对本作进行风格定调。

## 一、核心风格要素

### 1.1 叙事语调
- **冷幽默**：叙述者的声音不是严肃的全知者，而是带着一点旁观式的冷幽默。该沉重的地方沉重，但在沉重与沉重之间，要有让读者喘气的俏皮话。
- **反英雄感**：主角不是天生的英雄。他会怕、会懒、会在做决定之前犹豫很久。他的"英雄时刻"之所以动人，是因为他大部分时间看起来并不像英雄。
- **克制的煽情**：最感人的时刻往往用最平淡的语言写。苏棠不说"我担心你"，她说"你的抑盐针就剩一支了，浪费不起"。

### 1.2 章节标题
- 不做功能性描述（"海上来客""黑潮将至"太直白）
- 追求文学性/诗意/悬念感
- 可以用意象、对比、半句诗、角色台词、甚至反讽
- 例子参考：
  - "他比烟花寂寞" → 一个人的孤独
  - "弃我去者" → 告别
  - "暮色如血" → 末日氛围
  - "有拳头就要出拳" → 行动宣言

### 1.3 人物语言分化
- **顾北**：话多、嘴碎、爱骂人但每句话底下都是关心。常用反问句和反语。口头禅倾向："你他妈""卧槽""行行行"。
- **苏棠**：话少、精准、像手术刀。从不用感叹号。最严厉的批评用最平淡的语气说。偶尔会冒出一句让人意想不到的幽默。
- **老霍**：沉默型。一天说不了十句话，但每句都钉在点上。说话节奏慢，像他焊了三十年的铁。
- **罗峻**：军人腔，简短、准确、不带情绪。偶尔流露出来的情绪比长篇大论更有力量。
- **林渊**：失忆者的特殊视角——他看世界像是第一次看。别人习以为常的东西，在他眼里都是新鲜的、值得注意的。这给了叙事一种天然的"陌生化"效果。

### 1.4 日常密度
- 每个"大事件"之间，至少有一段纯日常的场景。不推进剧情，只推进人物关系。
- 吃饭、修东西、闲聊、发呆——这些场景不是浪费篇幅，是让角色"活"起来的呼吸空间。
- 猫腻的原则：读者记住的不是情节，是人。人是通过日常建立的。

### 1.5 世界观融入方式
- **绝不开"设定课"**。不允许出现"XXX是YYY的一种，它具有ZZZ的特征"这样的百科句式。
- 所有世界观信息都通过角色的日常行为和对话自然透出。
- 例：不说"盐化症是一种由海水中的盐毒引起的退行性疾病"——而是写"老霍举起自己的手，指关节发白，像涂了一层灰。'这就是盐化。不致命。但二十年后我的手就跟石头一样了。'"

## 三、风格改造要点清单

每章检查：
- [ ] 标题是否具有文学性/悬念感
- [ ] 是否有至少一处冷幽默或闲笔
- [ ] 人物对话是否有语气分化
- [ ] 世界观信息是否通过场景自然透出（而非讲解）
- [ ] 是否有"无事发生但在建立人物"的日常段落
- [ ] 最感人的时刻是否用了最克制的语言
